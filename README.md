# Resume project [January 2021]
Arduino Temp/humidity sensor 
Arduino IDE and UNO R3 microcontroller using C++
DHT11 sensor, 16x2 LCD screen,  220 ohm resistor

 LDC Display: 
  - temperature in Celsius and Fahrenheit 
  - Humidity in %rh
